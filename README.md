# Sistema de Gerenciamento de Campeões (Java)

Projeto desenvolvido em Java com foco em lógica de programação e orientação a objetos, simulando um sistema de gerenciamento de entidades inspirado em jogos.

## Objetivo

Praticar conceitos fundamentais de desenvolvimento backend, como:
- Programação Orientada a Objetos (POO)
- Manipulação de dados em memória
- Estruturação de regras de negócio
- Operações CRUD

## Tecnologias

- Java
- Estruturas de dados (listas)
- POO

## Funcionalidades

- Cadastro de campeões
- Listagem de campeões
- Busca por ID
- Remoção de campeões
- Geração de IDs únicos

## Estrutura

- Model → entidades
- Service/Manager → regras de negócio
- Main → execução do sistema

## Como executar

1. Clone o repositório:
git clone https://github.com/rafaelalves3070/lol-manager-java-project

2. Abra na sua IDE (IntelliJ, Eclipse ou VS Code)

3. Execute a classe `Main`
